## 课程信息与课程表位置对应
设计图-01
<div align=center>
<img src="screenshots/设计图-01.jpg" width="60%">
</div>

前提:

- 需要先将每一个课程表UI控件，以字典方式对应，如周一第一、二节的key："week1_12"，value: 对应位置的UI控件。


定位过程:
1. 从课程信息中拼凑key，eg"week1_12"。
2. 从字典中获取（定位）对应的UI控件。
3. 将课程的详细信息在对应UI控件中设置。


## 事件触发更新UI显示内容
- 每一个界面都是一个对象；
- 页面访问的第一个界面即为main入口；
- 通过绑定（链接）事件（函数）进行页面跳转；
- 明显可以感觉到UI开发明显不是面向过程。
